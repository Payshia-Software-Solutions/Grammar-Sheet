# **App Name**: GrammarWise

## Core Features:

- Homepage Implementation: Home page with a welcoming introduction, hero section, and clear call to action, faithfully reproduced from the Figma design.
- Lessons Index Page: Lessons page showcasing grammar lessons in a grid or list format based on the Figma design.
- Lesson Detail Pages: Detailed lesson pages explaining grammar rules with clear examples, all adhering to the provided Figma design.
- Interactive Quizzes: Interactive multiple-choice quizzes with immediate feedback for an engaging learning experience.
- Contact/About Page: Informative Contact/About page, presenting teacher information and a contact form according to the Figma design.
- Responsive Design: Fully responsive layout that adapts flawlessly to mobile, tablet, and desktop screens, ensuring an optimal viewing experience on any device.
- Consistent Navigation: Persistent navigation and footer elements to aid user experience and page consistency based on the design.

## Style Guidelines:

- Primary color: A soft, inviting blue (#64B5F6) evoking trust and learning.
- Background color: Light, desaturated blue (#E3F2FD), providing a calm backdrop.
- Accent color: A contrasting vibrant yellow (#FFD54F) for highlights and calls to action, enhancing the user interface.
- Body font: 'Inter', a grotesque sans-serif known for its modern and neutral appearance.
- Headline font: 'Space Grotesk', a proportional sans-serif for a techy, scientific feel in titles.
- Clean, consistent icons that visually represent grammar topics and enhance user navigation.
- A modern, grid-based layout with clear content hierarchy based on the figma design to create an intuitive and engaging user experience.
- Subtle transitions and feedback animations to enhance interactivity, without overwhelming the user.
